
= vpd home page
vpd is a a lightweight server-side application for playing videos. It relies on vlc for video rendering, while being
controlled by its network protocol.

== Installing

No release is available for the moment. You may consider installing from the sources.

== Getting help

* Read the documentation and the manpage
* Check the log file (--verbose option)
* Make sure you have the latest version of vpd
* Submit a bug report
* Contact us on IRC
